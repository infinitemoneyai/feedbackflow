How it connects

  • Screenshot button → captures iframe content → opens slide-in panel (same fields as widget)
  • Submit → creates feedback in same project inbox → same AI triage, same integrations
  • Share button → generates magic link → email-gated with optional password
  • External reviewer → enters email + password → gets the same viewer → feedback tagged with their email
  • All feedback → flows into existing project inbox, indistinguishable from widget-submitted feedback
